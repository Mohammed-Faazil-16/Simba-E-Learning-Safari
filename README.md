# 🦁 Simba E‑Learning Safari

**Simba E-Learning Safari** is a fun and interactive learning platform for kids. It includes math games, educational videos, music, a chatbot, and a colorful dashboard — all designed to make learning exciting and enjoyable.

---

## ✨ Features

- 🎮 **Math Games** – Addition, Subtraction, Multiplication, Division
- 📹 **Video Learning** – Watch and learn math concepts
- 🎼 **Music Room** – Listen to classical music
- 🤖 **Chatbot** – Ask Simba questions and learn
- 📊 **Dashboard** – Track progress with visual charts
